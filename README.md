# Dev Detective 🔎

Dev Detective is a GitHub profile finder built using HTML, CSS and JavaScript. It uses the GitHub REST API to fetch user information and display it in a clean and responsive interface.

The project also includes a Battle Mode where two GitHub users can be compared based on the total stars received across all of their public repositories.

---

## Features

- Search any GitHub user
- View profile information
- Display avatar, bio and join date
- Show followers, following and public repositories
- View latest 5 repositories
- Click repository links to open them on GitHub
- Loading state while fetching data
- Error handling for invalid usernames
- Battle Mode using Promise.all()
- Compare users based on total GitHub stars
